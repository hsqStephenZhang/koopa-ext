# Koopa-ext

this crate offers a wide range of utilities missed by koopa, including 

- graph
  - graph abstraction
  - traversal(PO, RPO)
  - dom tree construction
  - loop analysis
  - graphviz visualization
- passes
  - Simplify CFG
  - GVN
  - SCCP
  - Inline
  - Loop Simplify
  - LICM
- Misc
  - scalar evoution analysis
